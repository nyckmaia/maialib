◆ getChords()

std::vector< std::pair< int, Chord > > Score::getChords ( nlohmann::json  config = {})

Get the vertical stacked chords from the score object.

Parameters
config(Optional) A JSON object that contains:

partNames: list of strings
measureStart: integer
measureEnd: integer
minStack: integer
maxStack: integer
minDuration: string
maxDuration: string
continuosMode: boolean
includeDuplicates: boolean
includeUnpitched: boolean

Example:
{
"partNames": ["Violino", "Viola", "Violoncelo"],
"measureStart": 4,
"measureEnd": 10,
"minStack": 3,
"maxStack": 5,
"minDuration": "quarter",
"maxDuration": "whole",
"continuosMode": true,
"includeDuplicates": false,
"includeUnpitched": false
}

continuosMode
When continuosMode is true the algorithm will detect all vertical chords, including counterpoint notes
When continuosMode is false the algorithm will detect only the chords formed by the same attack notes, excluding counterpoint

includeDuplicates
When includeDuplicates is true the return chords can have multiple notes with the same pitch value (duplicate notes)
Example: ["C4", "C4", "E4", "G4"]
When includeDuplicates is false each chord will be filtered in post processing, returning the same chord but removing the duplicate pitch notes
Example: The original chord ["C4", "C4", "E4", "G4"] will be returned as ["C4", "E4", "G4"]

includeUnpitched
Returns
A list of pairs: {measureId, Chord object}

◆ instrumentFragmentation()

nlohmann::json Score::instrumentFragmentation ( nlohmann::json  config = nlohmann::json())

Get the called score 'instrument fragmentation' data.

This method is used only to get the input data to a specific Python plot graph.
The final Python plot shows the score 'instrument fragmentation' analysis graphically

Paper: Uma análise da organização e fragmentação de Farben de Arnold Schoenberg
Author: Igor Leão Maia (UNICAMP)
C++/Python Implementation by: Prof. Adolfo Maia Junior (UNICAMP)

To get more information about it:
https://www.researchgate.net/publication/321335427_Uma_analise_da_organizacao_e_fragmentacao_de_Farben_de_Arnold_Schoenberg
